Code value 128 is the midpoint of the 8-bit range. The sRGB transfer function decodes it to 21.6% luminance. Arithmetic on encoded values treats it as 50.2% — an error of 28.6 percentage points. The bottom strip shows what a perceptually correct (linear) midpoint actually looks like.
