WordPress deployment: upload all six .html files to your server (e.g. /wp-content/demos/ or a static subdirectory). In each post, use a Custom HTML block and paste the iframe code above, adjusting the src path to match your upload location. Gutenberg strips inline <script> tags from HTML blocks, so the iframe approach is required. All demos are fully self-contained (no external dependencies other than a Google Fonts call which degrades gracefully to Courier New).
